# Urbanix • Plataforma de Mapas e Inteligência Territorial

> Repositório público (sanitizado). Para dados sensíveis/privados, use os repositórios internos.

## O que é
A Urbanix integra dados georreferenciados e análises para apoiar decisões em planejamento urbano, desenvolvimento econômico, educação e expansão territorial.

## Principais capacidades
- Visualização em mapas (WebGIS)
- Grade territorial (leitura por quadra/bairro)
- KPIs e painéis executivos
- Camadas temáticas (ex.: empresas/CNPJ, escolas, densidade/perfis)
- Metodologia de análise e recomendação

## Como navegar neste repositório
- `docs/` — documentação pública
- `examples/` — exemplos e demos (sem dados sensíveis)
- `templates/` — templates (issues/PRs/padrões)
